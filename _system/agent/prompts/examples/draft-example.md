# Example draft

This is what the agent's output looks like after a draft is written.

The file lands in `drafts/`. The operator reviews it. Once approved, it gets moved (manually or by a playbook) to `approved/`. Once shipped, it moves to `posted/` with the publish date prefixed to the filename.
